My Story

I started prototyping zombies mechanics in Overwatch Workshop to learn systems design under constraints. That project grew into a full co‑op experience with bespoke bosses, perks, and zone progression—and a modern website with comics, a wiki, and devlogs. I’m now focused on web design as a career path, using the project as a proving ground for UI/UX craft, performance, and content strategy.

Timeline

Phase 1 — Prototype

Core loop, enemy waves, buyable zones, early perks. Shared first playtest code.

Phase 2 — Polish & Website

Modern landing page, sticky glass nav, mobile UX, lore/comics, analytics.

Phase 3 — Multi‑Map Content

Unique bosses per map, teleporter/doors, performance tuning, accessibility passes.

Phase 4 — Career Focus

Packaging the project for web‑design roles: case study, component library, code samples.
